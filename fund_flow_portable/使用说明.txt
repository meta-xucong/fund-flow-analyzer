==========================================
盘前资金流向分析系统 v1.0 - 便携版
==========================================

【系统要求】
- Windows 10/11 64位
- 网络连接（用于获取股票数据）
- 约500MB磁盘空间（首次运行自动下载Python环境）

【使用方法】
1. 双击运行 "启动器.bat"
2. 首次运行会自动下载并配置Python环境（约需3-5分钟）
3. 等待显示 "服务已启动!"
4. 用浏览器访问: http://localhost:5000
5. 使用完毕后，在命令行窗口按 Ctrl+C 停止服务

【功能特点】
✓ 每日9:25自动推送资金流向分析报告
✓ 历史数据回测（自定义日期范围）
✓ 苹果风格磨砂玻璃UI界面
✓ 动量策略 + 反转策略选股
✓ 5日持有期收益跟踪

【目录说明】
- backend/     : 后端API代码
- frontend/    : 前端界面文件
- database/    : 数据库文件（自动创建）
- scheduler/   : 定时任务代码
- python/      : 嵌入式Python（自动下载）

【注意事项】
1. 首次启动需要下载Python环境，请保持网络畅通
2. 如果端口5000被占用，程序会提示是否释放
3. 数据保存在 database/fund_flow.db
4. 报告文件保存在 reports/ 目录

GitHub: https://github.com/meta-xucong/fund-flow-analyzer
