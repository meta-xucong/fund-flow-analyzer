# 进程退出说明

## 正常退出方式

### 方法1：在启动窗口按 Ctrl+C
1. 确保启动器的命令行窗口在前台
2. 按 `Ctrl+C` 组合键
3. 等待程序显示 "[*] 再见!"
4. 窗口会自动关闭

### 方法2：关闭命令行窗口
直接点击命令行窗口的 `X` 按钮关闭

## 如果端口仍被占用

如果程序异常退出导致5000端口仍被占用，可以：

### 方法1：使用 check_port.py
```bash
python check_port.py
# 按提示输入 y 释放端口
```

### 方法2：手动释放端口
打开任务管理器，结束所有 `python.exe` 进程

### 方法3：重启电脑
最简单有效的方式

## 技术说明

程序使用了 `psutil` 库来安全地管理进程：
- 退出时会获取自身PID
- 只终止占用5000端口的其他进程
- **不会杀死自身进程或其他无关进程**
