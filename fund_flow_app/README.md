# 盘前资金流向分析系统 v1.0.0

一款基于苹果设计风格（磨砂玻璃UI）的股票资金流向分析工具，支持每日自动推送和历史回测。

## 功能特性

### 1. 每日信息推送
- 每天9:25自动获取A股实时数据
- 生成包含市场情绪、板块排行、选股策略的完整报告
- 支持邮件/Telegram推送（需配置）

### 2. 历史数据回测
- 自定义日期范围回测
- 动量策略（追涨）+ 反转策略（抄底）
- 5日持有期收益跟踪
- 支持结果打包下载

### 3. 苹果风格UI
- 磨砂玻璃质感界面
- 流畅的动画效果
- 响应式布局

## 安装要求

- Python 3.8+
- Windows/macOS/Linux

## 安装步骤

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 启动应用
```bash
python run.py
```

3. 打开浏览器访问
```
http://localhost:5000
```

## 使用说明

### 仪表盘
- 查看系统状态
- 快速获取今日报告
- 查看最新报告预览

### 今日报告
- 查看实时市场情绪
- 板块强度排行TOP 10
- 动量策略选股（追涨）
- 反转策略选股（抄底）
- 操作建议和仓位指导

### 历史回测
1. 选择开始和结束日期
2. 点击"开始回测"
3. 等待回测完成
4. 查看统计结果或下载报告

### 设置
- 开启/关闭每日自动推送
- 配置推送渠道（邮件/Telegram）

## 数据源

- 股票实时数据：腾讯API
- 板块数据：申万行业(spot)
- 个股信息：AKShare

## 回测规则

- 选股时间：每日9:25
- 买入价格：9:30开盘价
- 持有期：5个交易日
- 收益计算：(卖出价-买入价)/买入价×100%
