# 快速开始指南

## 1. 安装依赖

确保已安装Python 3.8+，然后运行：

```bash
pip install -r requirements.txt
```

## 2. 启动应用

```bash
python run.py
```

## 3. 访问应用

打开浏览器，访问：
```
http://localhost:5000
```

或者直接打开 `frontend/index.html` 文件

## 4. 功能使用

### 获取今日报告
- 点击左侧菜单"今日报告"
- 点击"刷新"按钮获取最新数据

### 运行回测
- 点击左侧菜单"历史回测"
- 选择开始和结束日期
- 点击"开始回测"
- 等待进度完成
- 下载回测报告

### 设置自动推送
- 点击左侧菜单"设置"
- 开启"启用每日推送"
- 配置推送渠道

## 5. 注意事项

- 首次启动会自动创建数据库
- 需要网络连接获取股票数据
- 回测需要一定时间，请耐心等待
- 数据保存在 `database/` 和 `reports/` 目录

## 6. 常见问题

**Q: 端口被占用？**
A: 修改 `run.py` 中的端口号

**Q: 无法获取数据？**
A: 检查网络连接，确保能访问腾讯API

**Q: 回测很慢？**
A: 这是正常的，获取历史数据需要时间

## 7. 技术支持

如有问题，请检查：
1. Python版本是否正确
2. 依赖是否完整安装
3. 网络连接是否正常
