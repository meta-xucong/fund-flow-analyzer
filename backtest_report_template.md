## 二、回测验证方法

### 方法A: 当日收益率验证 (推荐)

**买入点**: 9:25开盘价
**卖出点**: 当日收盘价

计算公式:
```
当日收益率 = (收盘价 - 开盘价) / 开盘价 × 100%
```

**评估标准**:
- ✅ 收益率 > 0%: 预测正确
- ❌ 收益率 ≤ 0%: 预测错误

### 方法B: 盘中机会验证

**买入点**: 9:25开盘价  
**观察点**: 当日最高价

计算公式:
```
盘中最大收益 = (最高价 - 开盘价) / 开盘价 × 100%
盘中最大回撤 = (最低价 - 开盘价) / 开盘价 × 100%
```

**评估标准**:
- 最大收益 > 2%: 存在获利了结机会
- 最大回撤 < -3%: 风险控制良好

### 方法C: 次日开盘验证 (短线)

**买入点**: T日9:25开盘价  
**卖出点**: T+1日9:25开盘价

计算公式:
```
隔夜收益率 = (T+1开盘价 - T日开盘价) / T日开盘价 × 100%
```

## 四、预期分析维度

### 1. 胜率统计
```
胜率 = 盈利次数 / 总交易次数 × 100%

评估:
- > 70%: 策略优秀，可实盘
- 50-70%: 策略一般，需优化
- < 50%: 策略失效，需调整
```

### 2. 盈亏比
```
盈亏比 = 平均盈利 / |平均亏损|

评估:
- > 2:1: 正期望值策略
- 1.5-2:1: 可接受
- < 1.5:1: 风险收益比不佳
```
